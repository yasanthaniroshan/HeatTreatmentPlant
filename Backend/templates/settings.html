System Information

Hardware Configuration

Water Temperature Sensor
DS18B20 Digital Temperature Sensor - Digital Pin 6
Air Sensor
DHT22 - Temperature & Humidity Sensor - Digital Pin 7
Heater Control
Relay-controlled heating element - Digital Pin 8
Cooling Control
Relay-controlled fan/cooling element - Digital Pin 9
Display
I2C LCD 16x2 - I2C Address 0x27

Control Algorithm

PID Control System

Control Method
PID (Proportional-Integral-Derivative) feedback control with time-proportional output modulation
Control Window
5-second time window for proportional output control
Sensor Reading Interval
Sensor readings and system updates occur every 2 seconds
Communication Protocol
Serial Communication at 9600 baud with Raspberry Pi
Data Format
Sensor data sent as JSON: {WA:temp, AI:temp, HU:humidity, SP:setpoint, PWR:output}
Commands received as: SET:temperature

Safety Information

Temperature Limits
The setpoint is limited between {{ min_setpoint }}°C and {{ max_setpoint }}°C to ensure safe system operation and prevent equipment damage.
Relay Safety
Relays are active-LOW configuration. Both heater and fan default to OFF (HIGH) for safe startup.
System Monitoring
The system continuously monitors sensor readings and maintains real-time feedback through the dashboard. Any system anomalies should be investigated immediately.
